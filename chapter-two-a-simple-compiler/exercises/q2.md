### Question No. 2

Sometimes it is necessary to modify the syntax of a programming lan-
guage. This is done by changing the CFG that the language uses. What
changes would have to be made to ac’s CFG (Figure 2.1) to implement
the following changes?
*  All ac programs must contain at least one statement.
*  All integer declarations must precede all ﬂoat declarations.
*  The ﬁrst statement in any ac program must be an assignment state-
ment.

### Solution

To require at least one statement, the empty production for Stmts must be removed. One possible modification is:
```C
Stmts → Stmt MoreStmts
MoreStmts → Stmt MoreStmts | λ
```
This guarantees that every program contains at least one statement.


To require all integer declarations to precede float declarations, declarations can be separated into two groups:
```C
Dcls → IntDcls FloatDcls
IntDcls → intdcl id IntDcls | λ
FloatDcls → floatdcl id FloatDcls | λ
```
This enforces that integer declarations appear before any float declarations.

To require the first statement to be an assignment statement, the grammar can distinguish the first statement from the remaining statements:
```
Prog → Dcls FirstStmt RestStmts $
FirstStmt → id assign Val Expr
RestStmts → Stmt RestStmts | λ
```
This guarantees that the first executable statement is an assignment.
