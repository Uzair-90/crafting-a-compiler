### **Question No. 1**

The CFG shown in Figure 2.1 deﬁnes the syntax of ac programs. Explain
how this grammar enables you to answer the following questions.
*  Can an ac program contain only declarations (and no statements)?
*  Can a print statement precede all assignment statements?

### Solution

1. Yes. The grammar allows programs containing only declarations because the nonterminal Stmts can derive λ (the empty string). Therefore, the production Prog → Dcls Stmts $ permits programs with declarations and no statements.

2. Yes. The grammar allows print statements to precede assignment statements because Stmt may derive either print id or id assign Val Expr, and the production Stmts → Stmt Stmts imposes no ordering restrictions on different kinds of statements.
