### Question No. 15

Show the NFA that would be created for the following expression using
the techniques of Section 3.8:
```(ab* c) | (abc* )``` Using MakeDeterministic, translate the NFA into a DFA. Using the
techniques of Section 3.8.3, optimize the DFA you created into a minimal
state equivalent.

### Solution

### Exercise 15

Regular expression:

```txt
(ab*c) | (abc*)
```

This means:

```txt
ab*c  = a, then zero or more b's, then c
abc*  = a, then b, then zero or more c's
```

Examples accepted:

```txt
ac
abc
abbc
abbbc
ab
abcc
abccc
```

Exercise 15 asks us to create the NFA, convert it to a DFA, and then minimize the DFA. 

### 1. NFA idea

```txt
                 b*
              +------+
              |      v
Start -> q0 -> q1 -> q2 -> q3*
          a     b     c

Start -> q0 -> q4 -> q5 -> q6*
          a     b     c*
                      ^ 
                      |
                      c
```

More simply:

```txt
Start
  |
  a
  v
 q1
 / \
c   b
|   |
v   v
q2* q3*
     | \
     c  b
     |   \
     v    v
    q4*  q5
     |     |
     c     b
     |     |
     v     v
    q4*   q5
           |
           c
           v
          q2*
```

### 2. DFA transition table

Let the alphabet be:

```txt
{a, b, c}
```

Final states:

```txt
q2, q3, q4, q6
```

| State | a    | b    | c    | Meaning                    |
| ----- | ---- | ---- | ---- | -------------------------- |
| `q0`  | `q1` | dead | dead | start                      |
| `q1`  | dead | `q3` | `q2` | after `a`                  |
| `q2*` | dead | dead | dead | accepted `ac` or `abb...c` |
| `q3*` | dead | `q5` | `q4` | accepted `ab`              |
| `q4*` | dead | dead | `q4` | accepted `abc*`            |
| `q5`  | dead | `q5` | `q6` | after `abb...`             |
| `q6*` | dead | dead | dead | accepted `abb...c`         |
| dead  | dead | dead | dead | invalid path               |

### 3. Minimized DFA

States `q2` and `q6` are equivalent because both are final and both go to dead on every input. Merge them into one state `qF`.

Final states:

```txt
q3, q4, qF
```

| State | a    | b    | c    |
| ----- | ---- | ---- | ---- |
| `q0`  | `q1` | dead | dead |
| `q1`  | dead | `q3` | `qF` |
| `q3*` | dead | `q5` | `q4` |
| `q4*` | dead | dead | `q4` |
| `q5`  | dead | `q5` | `qF` |
| `qF*` | dead | dead | dead |
| dead  | dead | dead | dead |

This minimized DFA recognizes exactly:

```txt
(ab*c) | (abc*)
```
