### Solution

### Exercise 23

Write a **Lex program** that counts word frequencies.

```lex
%{
#include <stdio.h>
#include <string.h>
#include <ctype.h>

#define MAXWORDS 10000
#define MAXLEN 100

typedef struct {
    char word[MAXLEN];
    int count;
} Entry;

Entry table[MAXWORDS];
int nwords = 0;

void add_word(char *s) {
    int i;

    /* convert to lowercase */
    for (i = 0; s[i]; i++)
        s[i] = tolower(s[i]);

    for (i = 0; i < nwords; i++) {
        if (strcmp(table[i].word, s) == 0) {
            table[i].count++;
            return;
        }
    }

    strcpy(table[nwords].word, s);
    table[nwords].count = 1;
    nwords++;
}
%}

%%

[A-Za-z]+      { add_word(yytext); }

[ \t\n]+       { /* ignore whitespace */ }

.              { /* ignore punctuation */ }

%%

int main() {
    int i;

    yylex();

    for (i = 0; i < nwords; i++) {
        printf("%s %d\n", table[i].word, table[i].count);
    }

    return 0;
}
```

### How it works

```txt
[A-Za-z]+
```

matches each word.

```txt
[ \t\n]+
```

ignores whitespace.

```txt
.
```

ignores punctuation and other single characters.

The Lex chapter explains that Lex rules have the form:

```txt
regular expression    action
```

and the action is executed when the regex matches the input.
