### Question No. 29

Let D be any deterministic ﬁnite automaton. Assume that you know D
contains exactly n states and that it accepts at least one string of length n
or greater. Show that D must also accept at least one string of length 2n
or greater.

### Exercise 29

Given:

```txt
D has exactly n states.
D accepts at least one string of length n or greater.
```

We must show:

```txt
D also accepts at least one string of length 2n or greater.
```

### Proof

Let the accepted string be:

```txt
w
```

and suppose:

```txt
|w| ≥ n
```

When the DFA reads `w`, it visits states like this:

```txt
q0, q1, q2, ..., q|w|
```

Since `|w| ≥ n`, the DFA visits at least `n + 1` state positions.

But the DFA has only `n` states.

So by the pigeonhole principle, some state must repeat:

```txt
qi = qj
```

where:

```txt
0 ≤ i < j ≤ |w|
```

That means the part of the string between positions `i` and `j` forms a loop.

So write:

```txt
w = xyz
```

where:

```txt
x takes D from q0 to qi
y takes D from qi back to qi
z takes D from qi to a final state
```

Since `y` is a loop:

```txt
xy^kz
```

is accepted for every `k ≥ 0`.

Now choose `k` large enough that:

```txt
|xy^kz| ≥ 2n
```

Therefore, `D` accepts at least one string of length `2n` or greater.

### Conclusion

Any DFA with `n` states that accepts one string of length at least `n` must contain a loop on an accepting path. That loop can be repeated to create accepted strings of arbitrarily large length, including length at least `2n`.
