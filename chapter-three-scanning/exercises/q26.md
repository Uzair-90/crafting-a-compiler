### Question No. 26

Let AllButLast be a function that returns all of a string but its last character.
For example, AllButLast(abc) = ab. AllButLast(λ) is undeﬁned. Let R be
any regular expression that does not generate λ. AllButLast(R) is the set
of strings denoted by R, with AllButLast applied to each string. Thus,
AllButLast(a+ b) = a+ . Show that AllButLast(R) is a regular set.

### Exercise 26

Define:

```txt
AllButLast(R) = { x | ∃ character c such that xc ∈ R }
```

Meaning: take every string in `R`, remove its **last character**, and collect the results.

### Claim

If `R` is regular, then `AllButLast(R)` is also regular.

### Proof idea using DFA

Since `R` is regular, there is a DFA:

```txt
D = (Q, Σ, δ, q0, F)
```

that recognizes `R`.

We build a new DFA `D'` that accepts a string `x` if `x` can reach some state from which **one more character** would enter a final state.

So the new final states are:

```txt
F' = { q ∈ Q | ∃ c ∈ Σ such that δ(q, c) ∈ F }
```

The transition function stays the same.

### Why this works

If after reading `x`, the DFA is in state `q`, and there exists some character `c` such that:

```txt
δ(q, c) ∈ F
```

then:

```txt
xc ∈ R
```

So:

```txt
x ∈ AllButLast(R)
```

### Conclusion

```txt
AllButLast(R) is regular.
```

Because we can construct a DFA for it by changing only the accepting states.
