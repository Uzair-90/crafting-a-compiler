### Question No. 21

You have scanned an integer literal into a character buﬀer (perhaps
yytext). You now want to convert the string representation of the literal
into numeric (int) form. However, the string may represent a value too
large to be represented in int form. Explain how to convert a string
representation of an integer literal into numeric form with full overﬂow
checking.


### Solution

### Exercise 21

Problem: after scanning an integer literal into `yytext`, convert it to `int`, but detect overflow. 

Use this safe method:

```c
int value = 0;

for each digit d in yytext:
    digit = d - '0';

    if value > (INT_MAX - digit) / 10:
        report overflow error;

    value = value * 10 + digit;
```

Why this works:

Before doing:

```c
value = value * 10 + digit;
```

we check whether that operation would exceed `INT_MAX`.

The important condition is:

```txt
value * 10 + digit <= INT_MAX
```

Rearranged safely:

```txt
value <= (INT_MAX - digit) / 10
```

So if:

```txt
value > (INT_MAX - digit) / 10
```

then the next digit would overflow.

Example with 32-bit signed `int`:

```txt
INT_MAX = 2147483647
```

Input:

```txt
2147483648
```

The scanner detects overflow before producing the final numeric value.

Final answer:

```txt
Convert digit by digit, and before multiplying by 10 and adding the new digit,
check whether the result would exceed INT_MAX.
```
