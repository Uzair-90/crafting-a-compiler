### Exercise 10

Given Grammar:

```txt
Prog     → Dcls Stmts $

Dcls     → Dcl Dcls
         | λ

Dcl      → floatdcl id
         | intdcl id

Stmts    → Stmt Stmts
         | λ

Stmt     → id assign Val ExprTail
         | print id

ExprTail → plus Val ExprTail
         | minus Val ExprTail
         | λ

Val      → id
         | num
```

### FIRST Sets

#### FIRST(Dcl)

```txt
Dcl → floatdcl id
    | intdcl id
```

```txt
FIRST(Dcl) = { floatdcl, intdcl }
```

---

#### FIRST(Dcls)

```txt
Dcls → Dcl Dcls
     | λ
```

```txt
FIRST(Dcls) = FIRST(Dcl) ∪ { λ }
            = { floatdcl, intdcl, λ }
```

---

#### FIRST(Stmt)

```txt
Stmt → id assign Val ExprTail
     | print id
```

```txt
FIRST(Stmt) = { id, print }
```

---

#### FIRST(Stmts)

```txt
Stmts → Stmt Stmts
      | λ
```

```txt
FIRST(Stmts) = FIRST(Stmt) ∪ { λ }
             = { id, print, λ }
```

---

#### FIRST(Val)

```txt
Val → id
    | num
```

```txt
FIRST(Val) = { id, num }
```

---

#### FIRST(ExprTail)

```txt
ExprTail → plus Val ExprTail
         | minus Val ExprTail
         | λ
```

```txt
FIRST(ExprTail) = { plus, minus, λ }
```

---

#### FIRST(Prog)

```txt
Prog → Dcls Stmts $
```

Since:

```txt
Dcls ⇒ λ
```

and:

```txt
Stmts ⇒ λ
```

the first symbol can come from:

```txt
Dcls
or
Stmts
or
$
```

Therefore:

```txt
FIRST(Prog) = { floatdcl, intdcl, id, print, $ }
```

---

### FOLLOW Sets

#### FOLLOW(Prog)

`Prog` is the start symbol.

```txt
FOLLOW(Prog) = { }
```

---

#### FOLLOW(Dcls)

From:

```txt
Prog → Dcls Stmts $
```

Add:

```txt
FIRST(Stmts) - {λ}
= { id, print }
```

Since:

```txt
Stmts ⇒ λ
```

also add:

```txt
$
```

Therefore:

```txt
FOLLOW(Dcls) = { id, print, $ }
```

---

#### FOLLOW(Dcl)

From:

```txt
Dcls → Dcl Dcls
```

Add:

```txt
FIRST(Dcls) - {λ}
= { floatdcl, intdcl }
```

Since:

```txt
Dcls ⇒ λ
```

also add:

```txt
FOLLOW(Dcls)
= { id, print, $ }
```

Therefore:

```txt
FOLLOW(Dcl)
= { floatdcl, intdcl, id, print, $ }
```

---

#### FOLLOW(Stmts)

From:

```txt
Prog → Dcls Stmts $
```

The symbol following `Stmts` is:

```txt
$
```

Therefore:

```txt
FOLLOW(Stmts) = { $ }
```

---

#### FOLLOW(Stmt)

From:

```txt
Stmts → Stmt Stmts
```

Add:

```txt
FIRST(Stmts) - {λ}
= { id, print }
```

Since:

```txt
Stmts ⇒ λ
```

also add:

```txt
FOLLOW(Stmts)
= { $ }
```

Therefore:

```txt
FOLLOW(Stmt) = { id, print, $ }
```

---

#### FOLLOW(ExprTail)

From:

```txt
Stmt → id assign Val ExprTail
```

`ExprTail` occurs at the end of the production.

Therefore:

```txt
FOLLOW(Stmt)
⊆ FOLLOW(ExprTail)
```

Thus:

```txt
FOLLOW(ExprTail) = { id, print, $ }
```

---

#### FOLLOW(Val)

From:

```txt
Stmt → id assign Val ExprTail
```

Add:

```txt
FIRST(ExprTail) - {λ}
= { plus, minus }
```

Since:

```txt
ExprTail ⇒ λ
```

also add:

```txt
FOLLOW(Stmt)
= { id, print, $ }
```

Therefore:

```txt
FOLLOW(Val)
= { plus, minus, id, print, $ }
```

---

### Final Answer

```txt
FIRST(Prog)     = { floatdcl, intdcl, id, print, $ }

FIRST(Dcls)     = { floatdcl, intdcl, λ }

FIRST(Dcl)      = { floatdcl, intdcl }

FIRST(Stmts)    = { id, print, λ }

FIRST(Stmt)     = { id, print }

FIRST(ExprTail) = { plus, minus, λ }

FIRST(Val)      = { id, num }
```

```txt
FOLLOW(Prog)     = { }

FOLLOW(Dcls)     = { id, print, $ }

FOLLOW(Dcl)      = { floatdcl, intdcl, id, print, $ }

FOLLOW(Stmts)    = { $ }

FOLLOW(Stmt)     = { id, print, $ }

FOLLOW(ExprTail) = { id, print, $ }

FOLLOW(Val)      = { plus, minus, id, print, $ }
```
