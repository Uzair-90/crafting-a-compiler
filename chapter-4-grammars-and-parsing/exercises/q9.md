### Exercise 9

Given grammar:

```txt
S → a S e
  | B

B → b B e
  | C

C → c C e
  | d
```

### FIRST Sets

#### FIRST(C)

```txt
C → c C e
C → d
```

The first terminals that can appear are:

```txt
c
d
```

Therefore:

```txt
FIRST(C) = { c, d }
```

---

#### FIRST(B)

```txt
B → b B e
B → C
```

The first production contributes:

```txt
b
```

The second production contributes:

```txt
FIRST(C) = { c, d }
```

Therefore:

```txt
FIRST(B) = { b, c, d }
```

---

#### FIRST(S)

```txt
S → a S e
S → B
```

The first production contributes:

```txt
a
```

The second production contributes:

```txt
FIRST(B) = { b, c, d }
```

Therefore:

```txt
FIRST(S) = { a, b, c, d }
```

---

### FOLLOW Sets

#### FOLLOW(S)

Since `S` is the start symbol:

```txt
$ ∈ FOLLOW(S)
```

From:

```txt
S → a S e
```

the symbol following `S` is:

```txt
e
```

Therefore:

```txt
FOLLOW(S) = { e, $ }
```

---

#### FOLLOW(B)

From:

```txt
S → B
```

`B` occurs at the end of the production, so:

```txt
FOLLOW(S) ⊆ FOLLOW(B)
```

Thus:

```txt
{ e, $ } ⊆ FOLLOW(B)
```

From:

```txt
B → b B e
```

the symbol following `B` is:

```txt
e
```

Therefore:

```txt
FOLLOW(B) = { e, $ }
```

---

#### FOLLOW(C)

From:

```txt
B → C
```

`C` occurs at the end of the production, so:

```txt
FOLLOW(B) ⊆ FOLLOW(C)
```

Thus:

```txt
{ e, $ } ⊆ FOLLOW(C)
```

From:

```txt
C → c C e
```

the symbol following `C` is:

```txt
e
```

Therefore:

```txt
FOLLOW(C) = { e, $ }
```

---

### Final Answer

```txt
FIRST(S) = { a, b, c, d }
FIRST(B) = { b, c, d }
FIRST(C) = { c, d }

FOLLOW(S) = { e, $ }
FOLLOW(B) = { e, $ }
FOLLOW(C) = { e, $ }
```
