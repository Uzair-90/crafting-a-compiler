### Exercise 13

The purpose of this exercise is to show how **recursion in a CFG represents lists of arbitrary length**.

Given:

```txt
Options → Options Option
        | λ
```

the grammar generates:

```txt
λ
Option
Option Option
Option Option Option
...
```

which is equivalent to:

```txt
Option*
```

The recursive rule:

```txt
Options → Options Option
```

adds one more `Option` to the list, while:

```txt
Options → λ
```

terminates the list.

### Key Lesson

A finite grammar can generate an infinite number of strings by using recursion. This pattern is commonly used in programming-language grammars to represent:

```txt
Declaration lists
Statement lists
Parameter lists
Argument lists
```

The main idea is:

```txt
Recursion in a CFG ⇒ Variable-length sequences
```
