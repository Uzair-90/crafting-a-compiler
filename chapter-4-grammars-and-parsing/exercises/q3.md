### Exercise 3

The given grammar is:

```txt
S      → Number
Number → [ Sign ] [ Digs period ] Digs
Sign   → plus
       | minus
Digs   → digit { digit }
```

where:

```txt
[ X ] = optional X
{ X } = zero or more repetitions of X
```

To transform the grammar into a standard CFG, replace each optional construct with a new nonterminal that can derive either the construct or `λ`, and replace the repetition construct with a recursive nonterminal.

The resulting standard CFG is:

```txt
S → Number

Number → OptSign OptDecimal Digs

OptSign → Sign
        | λ

OptDecimal → Digs period
           | λ

Sign → plus
     | minus

Digs → digit MoreDigs

MoreDigs → digit MoreDigs
         | λ
```

The transformations applied are:

```txt
[ Sign ]         ⇒ OptSign → Sign | λ

[ Digs period ] ⇒ OptDecimal → Digs period | λ

{ digit }        ⇒ MoreDigs → digit MoreDigs | λ
```
