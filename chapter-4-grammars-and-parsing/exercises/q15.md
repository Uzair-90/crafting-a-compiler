### Exercise 15

Show that regular grammars and finite automata have equivalent power.

This means:

```txt
Every regular grammar can be converted into a finite automaton.
Every finite automaton can be converted into a regular grammar.
```

So both describe exactly the same class of languages:

```txt
regular languages
```

---

### 1. From Regular Grammar to Finite Automaton

Consider a right-linear grammar:

```txt
A → aB
A → a
A → λ
```

We can convert it into a finite automaton as follows:

```txt
Each nonterminal becomes a state.
The start symbol becomes the start state.
Each production becomes a transition.
```

Rules:

```txt
A → aB     becomes     A --a--> B

A → a      becomes     A --a--> FinalState

A → λ      means       A is an accepting state
```

Example:

```txt
S → aA
A → b
```

becomes:

```txt
S --a--> A --b--> FinalState
```

This automaton accepts:

```txt
ab
```

---

### 2. From Finite Automaton to Regular Grammar

For every state in the automaton, create one nonterminal.

If the automaton has a transition:

```txt
q1 --a--> q2
```

create the grammar production:

```txt
Q1 → a Q2
```

If a state is accepting, add:

```txt
Q → λ
```

Example automaton:

```txt
q0 --a--> q1
q1 --b--> q2
q2 is final
```

Regular grammar:

```txt
Q0 → a Q1
Q1 → b Q2
Q2 → λ
```

This grammar generates:

```txt
ab
```

---

### Conclusion

Regular grammars and finite automata are equivalent because they can be transformed into each other.

```txt
Regular Grammar ⇔ Finite Automaton
```

Therefore, both define exactly the regular languages.
