### Exercise 11

Using the standard CFG obtained in Exercise 3:

```txt
S → Number

Number → OptSign OptDecimal Digs

OptSign → Sign
        | λ

OptDecimal → Digs period
           | λ

Sign → plus
     | minus

Digs → digit MoreDigs

MoreDigs → digit MoreDigs
         | λ
```

### FIRST Sets

#### FIRST(Sign)

```txt
Sign → plus
     | minus
```

```txt
FIRST(Sign) = { plus, minus }
```

---

#### FIRST(OptSign)

```txt
OptSign → Sign
        | λ
```

```txt
FIRST(OptSign) = { plus, minus, λ }
```

---

#### FIRST(MoreDigs)

```txt
MoreDigs → digit MoreDigs
         | λ
```

```txt
FIRST(MoreDigs) = { digit, λ }
```

---

#### FIRST(Digs)

```txt
Digs → digit MoreDigs
```

```txt
FIRST(Digs) = { digit }
```

---

#### FIRST(OptDecimal)

```txt
OptDecimal → Digs period
           | λ
```

```txt
FIRST(OptDecimal) = { digit, λ }
```

---

#### FIRST(Number)

```txt
Number → OptSign OptDecimal Digs
```

Since:

```txt
FIRST(OptSign) = { plus, minus, λ }
FIRST(OptDecimal) = { digit, λ }
FIRST(Digs) = { digit }
```

```txt
FIRST(Number) = { plus, minus, digit }
```

---

#### FIRST(S)

```txt
S → Number
```

```txt
FIRST(S) = FIRST(Number)
         = { plus, minus, digit }
```

---

### FOLLOW Sets

#### FOLLOW(S)

Since `S` is the start symbol:

```txt
FOLLOW(S) = { $ }
```

---

#### FOLLOW(Number)

From:

```txt
S → Number
```

`Number` occurs at the end of the production.

Therefore:

```txt
FOLLOW(Number) = FOLLOW(S)
               = { $ }
```

---

#### FOLLOW(OptSign)

From:

```txt
Number → OptSign OptDecimal Digs
```

The symbols following `OptSign` are:

```txt
OptDecimal Digs
```

Since:

```txt
FIRST(OptDecimal Digs) = { digit }
```

```txt
FOLLOW(OptSign) = { digit }
```

---

#### FOLLOW(Sign)

From:

```txt
OptSign → Sign
```

`Sign` occurs at the end of the production.

Therefore:

```txt
FOLLOW(Sign) = FOLLOW(OptSign)
             = { digit }
```

---

#### FOLLOW(OptDecimal)

From:

```txt
Number → OptSign OptDecimal Digs
```

The symbol following `OptDecimal` is:

```txt
Digs
```

Since:

```txt
FIRST(Digs) = { digit }
```

```txt
FOLLOW(OptDecimal) = { digit }
```

---

#### FOLLOW(Digs)

From:

```txt
Number → OptDecimal Digs
```

`Digs` occurs at the end of the production.

Therefore:

```txt
FOLLOW(Number) ⊆ FOLLOW(Digs)
```

giving:

```txt
{ $ }
```

Also, from:

```txt
OptDecimal → Digs period
```

the symbol following `Digs` is:

```txt
period
```

Therefore:

```txt
FOLLOW(Digs) = { period, $ }
```

---

#### FOLLOW(MoreDigs)

From:

```txt
Digs → digit MoreDigs
```

`MoreDigs` occurs at the end of the production.

Therefore:

```txt
FOLLOW(Digs) ⊆ FOLLOW(MoreDigs)
```

Thus:

```txt
FOLLOW(MoreDigs) = { period, $ }
```

---

### Final Answer

```txt
FIRST(S)          = { plus, minus, digit }

FIRST(Number)     = { plus, minus, digit }

FIRST(OptSign)    = { plus, minus, λ }

FIRST(OptDecimal) = { digit, λ }

FIRST(Sign)       = { plus, minus }

FIRST(Digs)       = { digit }

FIRST(MoreDigs)   = { digit, λ }
```

```txt
FOLLOW(S)          = { $ }

FOLLOW(Number)     = { $ }

FOLLOW(OptSign)    = { digit }

FOLLOW(OptDecimal) = { digit }

FOLLOW(Sign)       = { digit }

FOLLOW(Digs)       = { period, $ }

FOLLOW(MoreDigs)   = { period, $ }
```
