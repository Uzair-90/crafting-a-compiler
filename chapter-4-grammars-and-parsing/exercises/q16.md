### Exercise 16

Devise an algorithm to detect nonterminals that are unreachable from the start symbol.

A nonterminal is **reachable** if it can appear in some sentential form derived from the start symbol. Otherwise, it is **unreachable**.

---

### Algorithm

1. Mark the start symbol as reachable.

2. Repeat the following until no new nonterminals are discovered:

   - For every reachable nonterminal `A`
   - Examine each production of the form:

     ```txt
     A → α
     ```

   - Mark every nonterminal appearing in `α` as reachable.

3. Any nonterminal not marked reachable at the end is unreachable.

---

### Example

Grammar:

```txt
S → A B
A → a
B → b
C → c
```

Start with:

```txt
Reachable = { S }
```

From:

```txt
S → A B
```

add:

```txt
A, B
```

Now:

```txt
Reachable = { S, A, B }
```

Examine:

```txt
A → a
B → b
```

No new nonterminals are found.

Therefore:

```txt
Reachable = { S, A, B }
```

and:

```txt
C
```

is unreachable.

---

### Complexity

If there are:

```txt
N nonterminals
P productions
```

the algorithm visits each production at most once.

Therefore:

```txt
Time Complexity = O(P)
```

---

### Key Idea

A nonterminal is reachable if there exists a derivation:

```txt
S ⇒* αAβ
```

Starting from the start symbol and repeatedly following productions allows us to find all such nonterminals. Any nonterminal never reached by this process is unreachable and can be removed from the grammar without changing the language.
