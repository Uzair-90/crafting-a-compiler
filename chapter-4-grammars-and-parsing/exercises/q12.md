### Exercise 12

Show that if a grammar contains both left-recursive and right-recursive productions for the same nonterminal, the grammar is ambiguous.

Consider the grammar:

```txt
A → A a
  | a A
  | a
```

This grammar contains:

```txt
Left recursion:  A → A a
Right recursion: A → a A
```

---

### First Derivation

```txt
A
⇒ A a
⇒ A a a
⇒ a a a
```

This corresponds to the grouping:

```txt
((a) a) a
```

---

### Second Derivation

```txt
A
⇒ a A
⇒ a a A
⇒ a a a
```

This corresponds to the grouping:

```txt
a (a (a))
```

---

### Observation

Both derivations produce the same terminal string:

```txt
aaa
```

However, they produce different derivation structures and different parse trees.

Therefore:

```txt
The grammar is ambiguous.
```

---

### Conclusion

When a grammar contains both:

```txt
A → A α
```

and:

```txt
A → β A
```

for the same nonterminal `A`, the same string can often be derived in multiple ways.

Left recursion naturally creates left-associative structures, while right recursion naturally creates right-associative structures. When both are present simultaneously, multiple parse trees may exist for the same string, making the grammar ambiguous.
