### Question No. 3
Extend the ac scanner ![image](../../images/fig-2.5.png) in the following ways:
*  (a) A ﬂoatdcl can be represented as either f or float, allowing a more Java-like syntax for declarations. 
* (b) An intdcl can be represented as either i or int. 
* (c) A num may be entered in exponential (scientiﬁc) form. That is, an ac num may be suﬃxed with an optionally signed exponent (1.0e10, 123e-22 or 0.31415926535e1).

### Solution

```C
function SCANNER() returns Token

    while s.PEEK() = blank do
        call s.ADVANCE()

    if s.EOF()
    then
        ans.type ← $
    else

        if s.PEEK() ∈ {0,1,...,9}
        then
            ans ← SCANDIGITS()

        else

            if s.PEEK() ∈ {a,b,...,z,A,B,...,Z}
            then
                lexeme ← READWORD()

                if lexeme = "f" or lexeme = "float"
                then
                    ans.type ← floatdcl

                else if lexeme = "i" or lexeme = "int"
                then
                    ans.type ← intdcl

                else if lexeme = "p" or lexeme = "print"
                then
                    ans.type ← print

                else
                    ans.type ← id
                    ans.val ← lexeme

            else

                ch ← s.ADVANCE()

                switch(ch)

                    case '='
                        ans.type ← assign

                    case '+'
                        ans.type ← plus

                    case '-'
                        ans.type ← minus

                    default
                        call LEXICALERROR()

    return(ans)

end



function READWORD() returns string

    lexeme ← ""

    while s.PEEK() ∈ {a,b,...,z,A,B,...,Z}
        lexeme ← lexeme + s.ADVANCE()

    return lexeme

end



function SCANDIGITS() returns Token

    lexeme ← ""

    while s.PEEK() ∈ {0,1,...,9}
        lexeme ← lexeme + s.ADVANCE()

    if s.PEEK() = '.'
    then
        lexeme ← lexeme + s.ADVANCE()

        while s.PEEK() ∈ {0,1,...,9}
            lexeme ← lexeme + s.ADVANCE()

        ans.type ← fnum

    else
        ans.type ← inum


    if s.PEEK() = 'e' or s.PEEK() = 'E'
    then
        lexeme ← lexeme + s.ADVANCE()

        if s.PEEK() = '+' or s.PEEK() = '-'
        then
            lexeme ← lexeme + s.ADVANCE()

        if s.PEEK() ∉ {0,1,...,9}
        then
            call LEXICALERROR()

        while s.PEEK() ∈ {0,1,...,9}
            lexeme ← lexeme + s.ADVANCE()

        ans.type ← fnum


    ans.val ← lexeme

    return(ans)

end

```
