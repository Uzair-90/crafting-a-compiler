### Question No. 25

Let Seq(x, y) be the set of all strings (of length 1 or more) composed of
alternating x’s and y’s. For example, Seq(a, b) contains a, b, ab, ba, aba, bab,
abab, baba, and so on.
Write a regular expression that deﬁnes Seq(x, y).
Let S be the set of all strings (of length 1 or more) composed of a’s, b’s,
and c’s that start with an a and in which no two adjacent characters are
equal. For example, S contains a, ab, abc, abca, acab, acac, . . . but not c, aa,
abb, abcc, aab, cac, . . . . Write a regular expression that deﬁnes S. You may
use Seq(x, y) within your regular expression if you wish.


### Solution

### Exercise 25

Given:

```txt
Seq(x, y) = all nonempty strings made by alternating x and y
```

Examples:

```txt
x
y
xy
yx
xyx
yxy
xyxy
yxyx
...
```

A regular expression for this is:

```txt
x(yx)*y? | y(xy)*x?
```

Equivalently:

```txt
(xy)*(x|λ) | (yx)*(y|λ)
```

This covers strings starting with `x` and strings starting with `y`.

---

Now define `S`:

```txt
S = strings over {a,b,c}
that start with a
and have no two adjacent equal characters
```

Examples:

```txt
a
ab
ac
aba
abc
abca
acab
acac
```

A good regular expression is:

```txt
a(Seq(b,c)? | b Seq(a,c)? | c Seq(a,b)?)*
```

But cleaner without optional nested notation:

```txt
a((b|c)(a|λ))* (λ | b Seq(a,c)? | c Seq(a,b)?)
```

An easier correct construction is to split by what happens after the initial `a`:

```txt
a ( b Seq(a,c)? | c Seq(a,b)? | λ )
```

where `Seq(x,y)?` means either `λ` or an alternating sequence of `x` and `y`.

So using only `Seq` notation:

```txt
a(λ | b(λ | Seq(a,c)) | c(λ | Seq(a,b)))
```

This works because:

```txt
After starting with a:
- if next is b, the rest must alternate between a and c
- if next is c, the rest must alternate between a and b
- or the string can stop at a
```
