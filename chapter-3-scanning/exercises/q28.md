### Question No. 28

Let s be a string. Deﬁne Insert(s) to be the function that inserts a # into
each possible position in s. If s is n characters long, then Insert(s) returns
a set of n + 1 strings (since there are n + 1 places, a # may be inserted in a
string of length n).
For example, Insert(abc) = { #abc, a#bc, ab#c, abc# }. Insert applied to a set
of strings is the union of Insert applied to members of the set. Thus,
Insert(ab, de) = { #ab, a#b, ab#, #de, d#e, de# }.
Let R be any regular set. Show that Insert(R) is a regular set.
Hint: Given an FA for R, construct one for Insert(R).

### Exercise 28

We need to prove:

```txt
If R is regular, then Insert(R) is regular.
```

`Insert(s)` means insert `#` into every possible position of string `s`.

Example:

```txt
Insert(abc) = { #abc, a#bc, ab#c, abc# }
```

The exercise asks us to show this by constructing an FA for `Insert(R)`. 

### Construction

Suppose `R` is recognized by a DFA:

```txt
D = (Q, Σ, δ, q0, F)
```

We build a new finite automaton `D'` that recognizes `Insert(R)`.

The idea is:

```txt
D' behaves like D,
but at exactly one position it is allowed to read # without changing the simulated state of D.
```

So `D'` needs to remember whether `#` has already been inserted.

Use two copies of every state in `D`:

```txt
(q, 0) = before reading #
(q, 1) = after reading #
```

Start state:

```txt
(q0, 0)
```

Final states:

```txt
{ (q, 1) | q ∈ F }
```

This means we accept only if:

```txt
1. exactly one # was read
2. the remaining non-# characters form a string accepted by R
```

### Transitions

For every normal character `a ∈ Σ`:

```txt
(q, 0) --a--> (δ(q, a), 0)
(q, 1) --a--> (δ(q, a), 1)
```

For the inserted character `#`:

```txt
(q, 0) --#--> (q, 1)
```

No `#` transition is allowed from `(q, 1)`.

### Why this works

If the input is:

```txt
ab#c
```

then the automaton simulates `D` on:

```txt
abc
```

The `#` is consumed once, but it does not advance the original DFA state.

So if:

```txt
abc ∈ R
```

then:

```txt
ab#c ∈ Insert(R)
```

### Conclusion

Since `D'` has finitely many states:

```txt
2 × |Q|
```

`Insert(R)` is regular.
